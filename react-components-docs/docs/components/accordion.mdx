# Accordion

접을 수 있는 콘텐츠 패널을 제공하는 컴포넌트입니다. 다크모드와 라이트모드를 모두 지원합니다.

## 기본 사용법

```jsx
import { Accordion } from 'react-common-components-library';

function AccordionExample() {
  // 기본 사용법
  const items = [
    {
      title: "섹션 1",
      content: "섹션 1의 내용입니다."
    },
    {
      title: "섹션 2",
      content: "섹션 2의 내용입니다."
    }
  ];

  return (
    <Accordion items={items} className="custom-accordion" />
  );
}
```

export const BasicAccordionExample = () => {
  const items = [

## 다중 선택 사용법

`allowMultiple` 속성을 사용하면 여러 항목을 동시에 펼칠 수 있습니다.

```jsx
<Accordion 
  items={items} 
  allowMultiple 
  defaultExpanded={[0]} 
/>
```

export const MultipleAccordionExample = () => {
  const items = [
    {
      title: "섹션 1",
      content: "섹션 1의 내용입니다. 여러 섹션을 동시에 열어볼 수 있습니다."
    },
    {
      title: "섹션 2",
      content: "섹션 2의 내용입니다. 다른 섹션이 열려 있어도 이 섹션을 열 수 있습니다."

      title: "섹션 3",

    }
  ];

  return (
    <div className="component-preview">
      <Accordion 
        items={items} 
        allowMultiple 
        defaultExpanded={[0]} 
      />
    </div>
  );

## 커스텀 스타일 예제

다양한 스타일 옵션을 통해 아코디언의 모양을 커스터마이징할 수 있습니다.

```jsx
<Accordion 
  items={items}
  titleClassName="custom-title"
  contentClassName="custom-content"
  itemClassName="custom-item"
  disableAnimation={true}
  itemGap={10}
/>
```

export const CustomStyledAccordionExample = () => {
  const items = [
    {
      title: "커스텀 스타일 섹션 1",
      content: "이 섹션은 커스텀 클래스를 사용하여 스타일이 적용되었습니다."
    },
    {
      title: "커스텀 스타일 섹션 2",
      content: "이 섹션은 애니메이션이 비활성화되어 있고, 항목 간 간격이 넓습니다."
    }
  ];

  return (
    <div className="component-preview">
      <Accordion 
        items={items}
        titleClassName="custom-title"
        contentClassName="custom-content"
        itemClassName="custom-item"
        disableAnimation={true}
        itemGap={10}
      />
    </div>
  );
};

<CustomStyledAccordionExample />

## 커스텀 아이콘 예제

자신만의 아이콘을 사용하여 아코디언의 모양을 변경할 수 있습니다.

```jsx
const customIcon = (isOpen) => (
  <span>{isOpen ? '📖' : '📘'}</span>
);

<Accordion 
  items={items}
  customIcon={customIcon}
/>
```

export const CustomIconAccordionExample = () => {
  const items = [
    {
      title: "커스텀 아이콘 섹션 1",
      content: "이 섹션은 커스텀 아이콘을 사용합니다."
    },
    {
      title: "커스텀 아이콘 섹션 2",
      content: "섹션이 열리고 닫힐 때 아이콘이 변경됩니다."
    }
  ];

  const customIcon = (isOpen) => (
    <span>{isOpen ? '📖' : '📘'}</span>
  );

  return (
    <div className="component-preview">
      <Accordion 
        items={items}
        customIcon={customIcon}
      />
    </div>
  );
};

<CustomIconAccordionExample />

## Props

테마 모드(라이트/다크)는 자동으로 도큐사우러스의 테마 설정을 따릅니다.

| 속성 | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `items` | `Array<{title: ReactNode, content: ReactNode}>` | 필수 | 아코디언 항목 목록 |
| `titleStyle` | `CSSProperties` | - | 제목 스타일 |
| `contentStyle` | `CSSProperties` | - | 내용 스타일 |
| `style` | `CSSProperties` | - | 컨테이너 스타일 |
| `className` | `string` | '' | 아코디언 컨테이너에 적용할 추가 CSS 클래스 |
| `titleClassName` | `string` | '' | 아코디언 제목에 적용할 추가 CSS 클래스 |
| `contentClassName` | `string` | '' | 아코디언 내용에 적용할 추가 CSS 클래스 |
| `itemClassName` | `string` | '' | 아코디언 항목에 적용할 추가 CSS 클래스 |
| `iconClassName` | `string` | '' | 아코디언 화살표 아이콘에 적용할 추가 CSS 클래스 |
| `customIcon` | `(isOpen: boolean) => ReactNode` | - | 커스텀 화살표 아이콘 컴포넌트 |
| `disableAnimation` | `boolean` | false | 애니메이션 비활성화 여부 |
| `allowMultiple` | `boolean` | false | 여러 항목을 동시에 열 수 있는지 여부 |
| `defaultExpanded` | `number \| number[]` | - | 기본적으로 펼쳐진 항목의 인덱스 또는 인덱스 배열 |
| `onItemClick` | `(index: number, isOpen: boolean) => void` | - | 항목을 클릭할 때 호출되는 함수 |
| `itemGap` | `number` | 1 | 항목 간 간격 (픽셀) |
| `itemStyle` | `CSSProperties` | - | 아코디언 항목 스타일 |
| `iconStyle` | `CSSProperties` | - | 아이콘 스타일 |

## 접근성

Accordion 컴포넌트는 WAI-ARIA 아코디언 패턴을 준수합니다:

- 키보드 탐색을 지원합니다 (Tab 키로 아코디언 항목 간 이동, Enter 또는 Space 키로 아코디언 토글).
- 적절한 ARIA 속성을 사용하여 스크린 리더 사용자를 위한 접근성을 보장합니다.
- 확장/축소 상태가 적절하게 표시됩니다.
